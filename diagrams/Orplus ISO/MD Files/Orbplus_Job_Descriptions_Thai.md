## 1. ประธานเจ้าหน้าที่บริหาร (CEO)

**ตำแหน่ง:** ประธานเจ้าหน้าที่บริหาร (Chief Executive Officer)
**แผนก:** ผู้บริหาร
**รายงานต่อ:** คณะกรรมการบริษัท
**ผู้ดำรงตำแหน่งปัจจุบัน:** คุณเคิร์ก

### สรุปหน้าที่
CEO รับผิดชอบทิศทางกลยุทธ์โดยรวม การพัฒนาธุรกิจ และการดำเนินงานประจำวันของ Orbplus โดยรวมบทบาทผู้นำระดับสูงกับการมีส่วนร่วมในงานขายและการพัฒนาผลิตภัณฑ์

### ความรับผิดชอบหลัก
- กำหนดและดำเนินการตามวิสัยทัศน์ พันธกิจ และเป้าหมายเชิงกลยุทธ์ของบริษัท
- นำความพยายามในการพัฒนาธุรกิจและการหาลูกค้าใหม่
- ดูแลทุกแผนกและให้แน่ใจว่าสอดคล้องกับวัตถุประสงค์ของบริษัท
- จัดการความสัมพันธ์กับลูกค้าหลัก พันธมิตร และผู้มีส่วนได้ส่วนเสีย
- ตัดสินใจระดับสูงเกี่ยวกับนโยบาย กลยุทธ์ และการดำเนินงาน
- รักษาสุขภาพทางการเงินและการเติบโตอย่างยั่งยืนของบริษัท
- เป็นตัวแทนบริษัทในการสื่อสารภายนอกและการเจรจา
- สร้างและรักษาวัฒนธรรมและค่านิยมของบริษัท

### คุณสมบัติ
- ปริญญาตรีด้านบริหารธุรกิจ วิทยาการคอมพิวเตอร์ หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์อย่างน้อย 10 ปีในอุตสาหกรรมเทคโนโลยี/ซอฟต์แวร์
- มีผลงานที่พิสูจน์ได้ในการพัฒนาธุรกิจและการเป็นผู้นำ
- เข้าใจโมเดลธุรกิจ SaaS เป็นอย่างดี
- ทักษะการสื่อสารและการเจรจาที่ยอดเยี่ยม
- ความสามารถในการคิดเชิงกลยุทธ์และการตัดสินใจ

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- การเติบโตของรายได้และกำไร
- อัตราการได้ลูกค้าใหม่และการรักษาลูกค้า
- ความพึงพอใจและการรักษาพนักงาน
- การบรรลุเป้าหมายเชิงกลยุทธ์

## 2. หัวหน้าฝ่ายผลิตภัณฑ์และเทคโนโลยี

**ตำแหน่ง:** หัวหน้าฝ่ายผลิตภัณฑ์และเทคโนโลยี
**แผนก:** Product & Technology
**รายงานต่อ:** CEO
**ผู้ดำรงตำแหน่งปัจจุบัน:** คุณณัฐวุฒิ

### สรุปหน้าที่
นำทีมพัฒนาผลิตภัณฑ์และเทคโนโลยีเพื่อส่งมอบซอฟต์แวร์คุณภาพสูง รับผิดชอบกลยุทธ์ด้านเทคนิค แผนพัฒนาผลิตภัณฑ์ และการส่งมอบโครงการตามกำหนด

### ความรับผิดชอบหลัก
- กำหนดและดำเนินการตามวิสัยทัศน์ผลิตภัณฑ์และแผนเทคโนโลยี
- นำและให้คำปรึกษาทีมพัฒนา (ทั้งภายในและ outsource)
- รับรองคุณภาพโค้ด ความปลอดภัย และแนวปฏิบัติที่ดี
- ประสานงานกับ Tech Lead/Architect ในการออกแบบระบบ
- จัดการไทม์ไลน์โครงการ ทรัพยากร และส่งมอบงาน
- ประเมินและนำเทคโนโลยีใหม่มาใช้ตามความเหมาะสม
- ทำงานร่วมกับ Business Analyst ในเรื่อง requirements
- รับรองความเสถียรและประสิทธิภาพของแพลตฟอร์ม

### คุณสมบัติ
- ปริญญาตรีด้านวิทยาการคอมพิวเตอร์หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์อย่างน้อย 7 ปีในการพัฒนาซอฟต์แวร์
- พื้นฐานทางเทคนิคที่แข็งแกร่งในการพัฒนาเว็บและมือถือ
- ประสบการณ์ในการจัดการทีมพัฒนา
- ความรู้ด้าน Agile/Scrum
- เข้าใจโครงสร้างพื้นฐานคลาวด์และ DevOps

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- การส่งมอบผลิตภัณฑ์ตรงเวลาและอยู่ในงบประมาณ
- ตัวชี้วัดการทำงานและประสิทธิภาพของระบบ
- คุณภาพโค้ดและอัตราบั๊ก
- ประสิทธิภาพและความพึงพอใจของทีม

## 3. Tech Lead / Architect (Outsource - TERA)

**ตำแหน่ง:** Tech Lead / สถาปนิกซอฟต์แวร์
**แผนก:** Product & Technology
**รายงานต่อ:** หัวหน้าฝ่ายผลิตภัณฑ์และเทคโนโลยี
**ผู้ดำรงตำแหน่งปัจจุบัน:** TERA (พันธมิตร Outsource)

### สรุปหน้าที่
ให้ความเป็นผู้นำทางเทคนิคและคำแนะนำด้านสถาปัตยกรรมแก่ทีมพัฒนา ออกแบบระบบซอฟต์แวร์ที่ขยายได้ ปลอดภัย และดูแลรักษาง่าย

### ความรับผิดชอบหลัก
- ออกแบบสถาปัตยกรรมระบบสำหรับผลิตภัณฑ์ SaaS
- กำหนดมาตรฐานทางเทคนิคและแนวทางการเขียนโค้ด
- ตรวจสอบโค้ดและรับรองมาตรฐานคุณภาพ
- ให้คำแนะนำนักพัฒนาเกี่ยวกับแนวปฏิบัติที่ดี
- ประเมินและแนะนำ technology stack
- แก้ไขปัญหาทางเทคนิคที่ซับซ้อน
- จัดทำเอกสารข้อกำหนดทางเทคนิคและสถาปัตยกรรม
- ประสานงานกับ DevOps ในกลยุทธ์การ deployment

### คุณสมบัติ
- ปริญญาตรีด้านวิทยาการคอมพิวเตอร์หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์การพัฒนาซอฟต์แวร์อย่างน้อย 8 ปี
- ความรู้ความชำนาญด้านสถาปัตยกรรมระบบและ design patterns
- ประสบการณ์กับ microservices, APIs และ cloud platforms
- ทักษะการแก้ปัญหาและวิเคราะห์ที่ดี
- ทักษะการสื่อสารที่ยอดเยี่ยม

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- คุณภาพและความสามารถในการขยายของสถาปัตยกรรมระบบ
- การจัดการ technical debt
- อัตราการตรวจสอบโค้ดสำเร็จ
- ประสิทธิภาพการถ่ายทอดความรู้

## 4. นักพัฒนาซอฟต์แวร์ (หัวหน้า)

**ตำแหน่ง:** นักพัฒนาซอฟต์แวร์ (หัวหน้า)
**แผนก:** Product & Technology
**รายงานต่อ:** หัวหน้าฝ่ายผลิตภัณฑ์และเทคโนโลยี
**ผู้ดำรงตำแหน่งปัจจุบัน:** คุณเคิร์ก (CEO - บทบาทคู่)

### สรุปหน้าที่
นำความพยายามในการพัฒนาซอฟต์แวร์ ประสานงานระหว่างทีม front-end, back-end และ mobile development

### ความรับผิดชอบหลัก
- นำ development sprints และการจัดสรรงาน
- พัฒนาและดูแลรักษาฟีเจอร์ซอฟต์แวร์
- ประสานงานระหว่างทีมย่อยการพัฒนา
- มีส่วนร่วมในการตรวจสอบโค้ด
- รับรองการปฏิบัติตามมาตรฐานการเขียนโค้ด
- ทำงานร่วมกับ QA ในกลยุทธ์การทดสอบ
- จัดทำเอกสารการ implement ทางเทคนิค
- สนับสนุนกระบวนการ deployment และ release

### คุณสมบัติ
- ปริญญาตรีด้านวิทยาการคอมพิวเตอร์หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์การพัฒนาซอฟต์แวร์อย่างน้อย 5 ปี
- ความชำนาญในภาษาโปรแกรมที่เกี่ยวข้อง
- ประสบการณ์กับ version control (Git)
- เข้าใจ CI/CD pipelines
- ทักษะการสื่อสารและความเป็นผู้นำที่ดี

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- อัตราการทำ sprint สำเร็จ
- ตัวชี้วัดคุณภาพโค้ด
- ประสิทธิภาพการทำงานร่วมกันของทีม
- ไทม์ไลน์การส่งมอบฟีเจอร์

## 5. นักพัฒนาซอฟต์แวร์ - Front-End

**ตำแหน่ง:** นักพัฒนาซอฟต์แวร์ - Front-End
**แผนก:** Product & Technology
**รายงานต่อ:** นักพัฒนาซอฟต์แวร์ (หัวหน้า)
**ผู้ดำรงตำแหน่งปัจจุบัน:** คุณอภิสิทธิ์

### สรุปหน้าที่
พัฒนาและดูแลรักษาส่วนติดต่อผู้ใช้สำหรับเว็บแอปพลิเคชัน โดยรับรองประสบการณ์ผู้ใช้ที่ดีที่สุดและการออกแบบที่ responsive

### ความรับผิดชอบหลัก
- พัฒนาส่วนติดต่อเว็บแบบ responsive โดยใช้ frameworks สมัยใหม่
- Implement การออกแบบ UI/UX จากข้อกำหนด
- ปรับปรุงประสิทธิภาพ front-end
- รับรองความเข้ากันได้กับหลายเบราว์เซอร์
- เชื่อมต่อกับ back-end APIs
- เขียน unit tests สำหรับ front-end components
- ทำงานร่วมกับนักออกแบบและนักพัฒนา back-end
- ดูแลเอกสาร front-end

### คุณสมบัติ
- ปริญญาตรีด้านวิทยาการคอมพิวเตอร์หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์การพัฒนา front-end อย่างน้อย 3 ปี
- ความชำนาญใน HTML, CSS, JavaScript
- ประสบการณ์กับ React, Vue หรือ Angular
- เข้าใจหลักการออกแบบแบบ responsive
- คุ้นเคยกับ RESTful APIs

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- ความถูกต้องของการ implement UI
- ประสิทธิภาพการโหลดหน้าเว็บ
- เวลาในการแก้ไขบั๊ก
- ตัวชี้วัดคุณภาพโค้ด

## 6. นักพัฒนาซอฟต์แวร์ - Back-End (Outsource - INTENSE)

**ตำแหน่ง:** นักพัฒนาซอฟต์แวร์ - Back-End
**แผนก:** Product & Technology
**รายงานต่อ:** นักพัฒนาซอฟต์แวร์ (หัวหน้า)
**ผู้ดำรงตำแหน่งปัจจุบัน:** INTENSE (พันธมิตร Outsource)

### สรุปหน้าที่
พัฒนาและดูแลรักษา server-side logic, ฐานข้อมูล และ APIs สำหรับแพลตฟอร์ม SaaS

### ความรับผิดชอบหลัก
- พัฒนาและดูแลรักษา RESTful APIs
- ออกแบบและปรับปรุง database schemas
- Implement business logic และการประมวลผลข้อมูล
- รับรองความปลอดภัยและการ authentication ของ API
- เขียน unit และ integration tests
- ปรับปรุงประสิทธิภาพเซิร์ฟเวอร์
- จัดทำเอกสารข้อกำหนด API
- สนับสนุนการ deployment และการแก้ไขปัญหา

### คุณสมบัติ
- ปริญญาตรีด้านวิทยาการคอมพิวเตอร์หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์การพัฒนา back-end อย่างน้อย 3 ปี
- ความชำนาญในภาษา server-side (Python, Node.js, Java ฯลฯ)
- ประสบการณ์กับฐานข้อมูล relational และ NoSQL
- เข้าใจหลักการออกแบบ API
- ความรู้ด้านแนวปฏิบัติที่ดีด้านความปลอดภัย

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- ประสิทธิภาพและความน่าเชื่อถือของ API
- คุณภาพโค้ดและ test coverage
- เวลาในการแก้ไขบั๊ก
- ความสมบูรณ์ของเอกสาร

### สรุปหน้าที่

## 14. Cloud/DevOps Engineer (Outsource - INET)

**ตำแหน่ง:** Cloud/DevOps Engineer
**แผนก:** Cloud & Infrastructure
**รายงานต่อ:** หัวหน้าฝ่ายผลิตภัณฑ์และเทคโนโลยี
**ผู้ดำรงตำแหน่งปัจจุบัน:** INET (พันธมิตร Outsource)

### สรุปหน้าที่
จัดการโครงสร้างพื้นฐานคลาวด์ deployment pipelines และรับรองความน่าเชื่อถือและความปลอดภัยของระบบ

### ความรับผิดชอบหลัก
- ออกแบบและดูแลโครงสร้างพื้นฐานคลาวด์
- Implement และจัดการ CI/CD pipelines
- ตรวจสอบประสิทธิภาพและความพร้อมใช้งานของระบบ
- จัดการการกำหนดค่าความปลอดภัยและการควบคุมการเข้าถึง
- ดำเนินการวางแผน backup และ disaster recovery
- ปรับปรุงต้นทุนและทรัพยากรคลาวด์
- สนับสนุนทีมพัฒนาในการ deployment
- จัดทำเอกสารโครงสร้างพื้นฐานและขั้นตอน

### คุณสมบัติ
- ปริญญาตรีด้านวิทยาการคอมพิวเตอร์หรือสาขาที่เกี่ยวข้อง
- ประสบการณ์ DevOps/คลาวด์อย่างน้อย 3 ปี
- ความชำนาญใน cloud platforms (AWS, Azure, GCP)
- ประสบการณ์กับ containerization (Docker, Kubernetes)
- ความรู้ด้าน infrastructure as code
- เข้าใจแนวปฏิบัติด้านความปลอดภัยที่ดี

### ตัวชี้วัดผลการปฏิบัติงาน (KPIs)
- Uptime และความพร้อมใช้งานของระบบ
- อัตราความสำเร็จในการ deployment
- เวลาตอบสนองต่อ incident
- ประสิทธิภาพต้นทุนโครงสร้างพื้นฐาน
