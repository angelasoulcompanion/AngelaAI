# MCP Servers for M4 (Work MacBook)

This folder contains MCP server configurations specific to M4.

## Setup Instructions

1. Copy servers from `mcp_servers_m3/` as needed
2. Update credentials in each server's `credentials/` folder
3. Update `.mcp.json` to point to `mcp_servers_m4/`
